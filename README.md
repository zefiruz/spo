## Запуск на локальной машине
1. Клонируем репозиторий  
  ```bash
  git clone https://github.com/zefiruz/spo.git
  ```

2. Переходим в spo
  ```bash
  cd spo
  ```

3. Устанавливаем зависимсти  
  ```bash
  npm install
  ```
  
4. Запускаем приложение  
  ```bash
  npm run dev
  ```
## Данные для пользователя администратор
Находяться в файле config.js  
По умолчанию:
```js
{
login: 'admin',
password: 'superpassword123'
}
```
